binarySearch

src/binarySearch.js

Performs a binary search of value in array

binarySearch(array: Array<number>, value: number, options: Any): number
Parameters
array (Array<number>) Array in which value will be searched. It must be sorted.
value (number) Value to search in array
options (Any)
Returns
number: If value is found, returns its index in array. Otherwise, returns a negative number indicating where the value should be inserted: -(index + 1)
